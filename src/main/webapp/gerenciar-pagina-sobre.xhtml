Sobre nosso sistema

Esse sistema imobiliário foi desenvolvido com o intúito de trabalhar o conteúdo da disciplina de Programação para Web 2 na prática.

Para a confecção do trabalho foram utilizadas as seguintes tecnologias:

  • Linguagem de Programação - Java 8
  • Java EE 7
  • HTML 5
  • CSS 4
  • Javascript
  • Primefaces 8.0
  • Tomcat 8.0.53
  • Java CDI - Weld 2.4.8
  • JPA - Hibernate 5.4.14
  • Mysql 8.0.20
  • FlywayDB 6.5.0
  • Lombok 1.18.12
